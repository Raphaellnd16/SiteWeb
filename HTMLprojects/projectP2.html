SPORTSNEWS

Développement Web : HTML • CSS • JavaScript • VS Code • Design Web

Contexte

Le projet consiste à créer un site web moderne et dynamique entièrement dédié à l’actualité sportive. L’objectif est de proposer une plateforme où les passionnés de sport peuvent suivre en temps réel les dernières nouvelles, résultats, analyses et interviews, le tout dans une interface claire, visuelle et engageante.
